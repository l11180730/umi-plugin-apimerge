See our website [@umijs/plugin-apimerge](https://umijs.org/plugins/plugin-model) for more information.

## Install

Using npm:

```bash
$ npm install --save-dev @umijs/plugin-apimerge
```

or using yarn:

```bash
$ yarn add @umijs/plugin-apimerge --dev
```
